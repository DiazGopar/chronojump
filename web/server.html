Reliability of the server

There are two levels of reliability:

  • General data Uploaded by anyone with the technology.
  • Reliable data Uploaded by anyone with the technology and accredited by SAFE research group (Universitat Ramon Llull) using the following directions:

Reliable data Profile Evaluator-trainer

  • National trainer
  • Current work: club, institution, category
  • Years as elite trainer
  • Historical on trained sports and specialties
  • Experience on using measurement equipment
  • Career (100 words)

Reliable data Profile Evaluator-researcher

  • Maximum reached degree
  • Teacher at University?
  • Sport science researches using measure
  • Other researches. Years as researcher
  • University research groups
  • Received grants
  • Career (100 words)

Reliable data Criteria by session

  • Informed consent
  • Warming up
  • Protocol variations (if any)
  • New tests protocols and validations (if any)
  • Data has been published? Where?
  • Justification of the research (100 words)

Reliable data Criteria by persons

  • Incomplete data of the persons
  • Person's height so low: < 0.5 m
  • Person's height so tall: > 2.1 m
  • Weight so low: < 30 Kg
  • Weight so high: > 200 Kg

Reliable data Criteria by jumps

  • Flight time too low for jumps without extra weight: < 0.3 s
  • Flight time too high: > 0.8 s
  • Contact time too low: < 0.15 s
  • Contact time too high for jumps without extra weight: > 0.4 s
  • Squat or Rocket higher than CMJ on the same persons
  • CMJ values higher that ABK on the same persons
  • Repeated values on jumps
  • Women results better than man results

All this items can be refined in the future.
If the evaluator has a mixed profile, it will need to answer both profiles.
Currently all the data is marked as General data. In the future, when there's more data, reliability checks will apply.
