Install Chronojump on Linux

  • Debian/Ubuntu: Open a terminal and paste this:
    sudo add-apt-repository ppa:ylatuya/ppa
    sudo apt-get update
    sudo apt-get install chronojump
  • Fedora: Soon available
  • Sources [expert]: Download
    ./configure, make, sudo make install
