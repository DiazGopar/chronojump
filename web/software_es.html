Instalar Chronojump en Linux

  • Debian/Ubuntu: Pegar este texto en una terminal:
    sudo add-apt-repository ppa:ylatuya/ppa
    sudo apt-get update
    sudo apt-get install chronojump
  • Fedora: Disponible en breve
  • Sources [experto]: Download
    ./configure, make, sudo make install
