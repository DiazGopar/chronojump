En este gráfico puede ver los diferentes evaluadores que han subido sus contribuciones. Cada evaluador dispone de dos barras, una para el número de saltos subidos y otra para el numero de sujetos añadidos.

Chronojump dispone de un servidor para compartir datos entre usuarios.
El servidor ya ha comenzado a recoger y mostrar datos gracias a las aportaciones de los usuarios.
Si hay nuevos datos, los gráficos se actualizan por la noche.
